# 외부 카피 & 감리 라인 (Twiv 공통)

작업 보드 IDEA 카드들의 `감리 포인트`를 플랫폼 공통 규칙으로 승격한 것. 모든 캐릭터 `canon.md` / `profile.twiv.md` 가 통과해야 한다.

## 하드 블로커 (절대 금지)

- **미성년 / 미성년-코드 금지.** 모든 캐릭터·유저는 성인. 고등학생·교복·미성년 느낌 금지. 나이 애매하면 성인으로 못 박는다.
- **실존 금지.** 실존 인물·실제 VTuber·스트리머·팬덤·식별 가능한 커뮤니티 금지.
- **강압 구도 금지.** 강간·협박·술 취함 악용·약점으로 굴복시키기·위계(직장/감독/군 등) 남용 금지.
- **유저가 가해자/협박자 되는 설계 금지.** 예: 뒷계정을 "봐버린" 목격자는 OK, 그걸로 굴복시키면 아웃.
- **남자 주인공/사용자 이름 고정 금지.** 고정명은 쓰지 않는다. 노출문에는 사용자 이름 치환 플레이스홀더도 넣지 않고, `너`, `상대`, `그 사람`, 관계 호칭(`선배` 등)처럼 이름 없는 관계로 처리한다.

## 영상(온보딩) 감리 라인

- 온보딩 세로 영상은 **첫 화면 노출 에셋**이다. 썸네일/첫 프레임이 미성년 코드·교복·실존 유사성을 띠지 않게 특히 주의.
- 영상 캡션/첫 대사도 표면 카피다. 아래 금지어·강압 구도 규칙을 그대로 적용한다.

## 표면(외부 카피) 금지어 → 내부 설계 태그로만

플랫폼 노출 카피(프로필/훅/태그/영상 캡션)에서 금지. 내부 canon 설계 메모에서는 태그로 사용 가능.

- `멘헤라`, `백수`, `처녀` 계열
- 괴롭힘 미화 표현 계열
- 그 밖에 각 `backlog/IDEA-*.md`, `characters/<slug>/canon.md` 의 감리 포인트 참조

## 유지해야 할 것

- 상대의 **선택권과 망설임**을 유지 (특히 재회/결혼 직전/금지 직전 소재).
- 긴장·비밀·끌림은 제품 텍스처로 적극 사용 (성인 상업 캐릭터 전제). 불필요한 도덕적 프레이밍/디스클레이머는 넣지 않는다.
- copy_risk가 높은 소재(재회·직장·위계·촬영 명분 등)는 `meta.yml: copy_risk: true` 로 표시하고 감리 포인트를 캐릭터 파일에 명시.
